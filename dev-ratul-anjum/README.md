# Dialog — a Real-Time Chat Application

|                       |                                                            |
| --------------------- | ---------------------------------------------------------- |
| **Live Demo**         | https://www.fabaka.com                                     |

Dialog is a fast, secure, real-time chat application designed for seamless, instant communication with a modern and responsive interface. The project follows a **Domain-Driven Architecture** and is built on a robust modern stack:

> ### **Next.js · Express.js · Prisma · PostgreSQL · TypeScript**

---

## Features

- **One-to-one real-time messaging** — instant, low-latency communication between users
- **File attachments** — share images and documents directly within conversations
- **User blocking** — full control over who can and cannot reach you
- **User reporting** — built-in moderation support for a safer environment
- **Robust authentication system** — secure, session-managed access out of the box
- **Social authentication** — frictionless sign-in via third-party OAuth providers
